# Museon
Multimedia and Web Design Course Project
# 🎨 Museon - Art Museum & Marketplace

# Team Member: Sevda Nur Uysal

## 🚀 Features
- 🖼️ 3D gallery with hover effects
- 📱 Mobile responsive
- ⚡ Fast loading
- 🎨 Modern design

# Frontend
HTML5 – structure of the website (pages, forms, layout)
CSS3 – styling, layout design, responsiveness, animations
JavaScript (Vanilla JS) – interactivity (modals, favorites, dynamic UI updates)
# UI & Styling Libraries
Font Awesome – icons (heart, user, trash, etc.)
Google Fonts (Inter) – modern typography for clean design
Custom CSS animations – hover effects, transitions, tiles animations
# Backend
PHP – server-side logic (login system, sessions, form handling)
MySQL (via XAMPP) – database for user authentication (my_auth_system)
PHP Sessions – to manage logged-in users and control access
# API Integration
Pexels API – used to fetch high-quality artwork/images dynamically
Fetch API (JavaScript) – used to send requests to Pexels API and retrieve data
# Data Storage
LocalStorage (Browser Storage) – used to store user favorites locally in JSON format
# Development Environment
XAMPP – local server environment (Apache + MySQL + PHP)
phpMyAdmin – database management interface
VS Code – code editor

◦ Step-by-step instructions to run the project locally
1- Install XAMPP
2- Start Apache + MySQL in XAMPP
3- Move project folder to htdocs
4- Open http://localhost/phpmyadmin
5- Create database: my_auth_system
6- Import/run required SQL tables
7- Check DB connection in PHP file
8- Open project: http://localhost/museon/
9- Test login, favorites, and API features
